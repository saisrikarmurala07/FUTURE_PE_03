# 🚀 AI SEO Blog Generator & Content Cluster System

## 📌 Overview

This project generates SEO-optimized blog content and content clusters using AI tools like ChatGPT, Claude, or Gemini.

It helps businesses:

* Rank on Google
* Target the right keywords
* Build topical authority
* Improve internal linking

---

## 🧰 Tools Used

* ChatGPT / Claude / Gemini
* Google Docs / MS Word
* GitHub

(Optional: Surfer SEO, Ahrefs free tools)

---

## 💡 Features

✔ Generate keyword intent (informational, transactional)
✔ Create SEO blog outlines (H1–H3)
✔ Write long-form blog posts
✔ Build content clusters (pillar + supporting topics)
✔ Add internal linking strategy
✔ Optimize for local SEO

---

## 📂 Project Structure

* prompts → AI prompt templates
* outputs → Sample generated blog
* docs → Prompt logic
* src → Python script

---

## ▶️ How to Use

### 1. Install Requirements

```bash
pip install -r requirements.txt
```

### 2. Run Generator

```bash
python src/generator.py
```

### 3. Customize Inputs

Edit:

* Keyword
* Business type
* Location
